Proceso de Desarrollo

• ¿Qué herramienta se utilizó?

Para el desarrollo del presente portal web se utilizó la herramienta de inteligencia artificial ChatGPT, empleada como apoyo para la creación del código HTML y CSS, el diseño visual de la interfaz, la organización del contenido y la implementación de elementos interactivos dentro del sitio web.

• ¿Qué se le pidió? (prompts)

Se solicitaron instrucciones relacionadas con la creación de un portal web moderno y profesional para la materia MOS. Entre las peticiones realizadas se incluyeron la creación de tarjetas para los integrantes, incorporación de animaciones visuales, diseño con colores morados y complementarios, integración de formularios de contacto, navegación interactiva, reproducción de audio, visualización del guion del podcast y aplicación de estilos avanzados mediante CSS.

• ¿Qué resultados se obtuvieron?

Como resultado se obtuvo un portal web interactivo, dinámico y visualmente atractivo, compuesto por diferentes secciones organizadas para la presentación del proyecto MOS. El sitio incluye animaciones, diseño responsivo, formularios funcionales, tarjetas informativas, reproducción multimedia y una estética moderna basada en efectos visuales tipo glassmorphism y combinaciones cromáticas armónicas.
